# Comparative Analysis of Machine Learning Models for Healthcare Diagnostics

## Description
This project presents a comparative study of ten machine learning algorithms for predictive modeling in healthcare diagnostics. Using five real medical datasets from Kaggle, we evaluated Logistic Regression, Decision Tree, Random Forest, Gradient Boosting, XGBoost, SVM, KNN, Naive Bayes, AdaBoost, and Bagging. Models were assessed using Accuracy, Precision, Recall, F1-Score, and ROC-AUC. Results are visualized with bar charts, heatmaps, radar charts, and ROC curves to highlight each algorithm’s strengths and weaknesses.

## Motivation
Machine learning has become an essential tool in medical decision support systems. Different algorithms perform differently depending on disease type and dataset characteristics. This study aims to identify the most effective models for predicting diseases and provide insights for future research.
